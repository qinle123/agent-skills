description: "Archive a completed OpenSpec change"

This is a subcommand alias for openspec-workflow.

## Instructions

The user wants to run the `archive` workflow for an OpenSpec change.

1. Load the `openspec-workflow` skill with `skill_view('openspec-workflow', file_path='references/opsx-archive.md')`
2. Follow its instructions step by step
3. The user's argument after `/openspec-archive` is the change name
