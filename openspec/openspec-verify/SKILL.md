This is a subcommand alias for openspec-workflow.

## Instructions

The user wants to run the `verify` workflow for an OpenSpec change.

1. Load the `openspec-workflow` skill with `skill_view('openspec-workflow', file_path='references/opsx-verify.md')`
2. Follow its instructions step by step
3. The user's argument after `/openspec-verify` is the change name
